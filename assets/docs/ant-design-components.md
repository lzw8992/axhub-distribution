## 使用建议

### 组件选择原则

1. **通用组件**：适用于所有页面，是最基础的 UI 元素
2. **布局组件**：用于构建页面整体结构
3. **导航组件**：用于页面间的跳转和定位
4. **数据录入组件**：用于表单和数据输入场景
5. **数据展示组件**：用于展示各类数据
6. **反馈组件**：用于操作反馈和状态提示
