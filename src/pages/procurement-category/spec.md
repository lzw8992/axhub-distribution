# 采购类别管理

## 📋 业务与功能

### 1.1 核心目标
采购类别管理页面用于维护企业采购系统中的商品/服务分类体系。支持多级分类结构，便于采购人员进行分类管理和统计分析。

### 1.2 功能清单

- **[查询筛选]**：支持按类别编码、类别名称进行模糊搜索、启用状态下拉选择
- **[列表展示]**：展示所有采购类别数据，包含编码、名称、父级、排序、状态、创建时间等字段
- **[新增类别]**：打开抽屉表单，填写类别信息（编码自增长、名称必填、父级可选、排序号）
- **[编辑类别]**：修改已有类别信息
- **[删除类别]**：删除单个类别（需确认）
- **[查看类别详情]**：点击采购类别名称，查看采购类别详情页
- **[启用/禁用]**：切换类别状态
- **[批量操作]**：批量启用、批量禁用、批量删除
- **[导入导出]**：支持 Excel 导入导出功能
- **[分页功能]**：支持分页展示，可调整每页条数

### 1.3 交互要点

- [筛选查询]：输入关键词后点击查询或按回车触发搜索
- [新增/编辑]：点击按钮打开抽屉表单，表单校验通过后保存
- [删除]：点击删除弹出确认对话框，确认后删除
- [启用/禁用]：点击状态切换按钮，即时生效
- [批量操作]：勾选表格行后，批量操作按钮可用
- [分页切换]：切换页码或每页条数时重新加载数据

### 2.2 数据来源

- **数据类型**：列表数据
- **数据源**：生成示例数据（后台/ToB 场景，真实、专业、完整）
- **关键字段**：
  - `id`: 主键 ID
  - `code`: 类别编码（自增长，格式：CG001）
  - `name`: 类别名称
  - `parentId`: 父级类别 ID（可选，null 表示顶级类别）
  - `parentName`: 父级类别名称
  - `sort`: 排序号（数字，越小越靠前）
  - `status`: 状态（1-启用，0-禁用）
  - `createTime`: 创建时间
  - `updateTime`: 更新时间

## 🎨 布局与结构

### 3.1 整体布局

- **布局模式**：单栏布局，内容区居中
- **容器宽度**：流式宽度，最大 1440px
- **关键尺寸**：
  - 页面内边距：24px
  - 卡片内边距：24px
  - 筛选区与表格间距：16px
  - 表格行高：48px

### 3.2 响应式适配

- **桌面端（≥1200px）**：完整展示所有列
- **平板端（768-1199px）**：隐藏部分次要列（创建时间、更新时间）
- **移动端（<768px）**：简化布局，操作按钮收起为下拉菜单
