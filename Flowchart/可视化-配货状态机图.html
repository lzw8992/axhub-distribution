📋 状态转换规则详解

📋 待配货状态

  • → 调拨中 (选择【调拨】)
  • → 延迟调拨 (选择【延迟调拨】)
  • → 采购中 (选择【采购】)
  • → 配货关闭 (关闭需求)
  • → 待配货 (转交任务-变更处理人)

🚚 调拨中状态

  • → 配货完成 (审批通过)
  • → 配货退回 (审批驳回-作废调拨单)

⏰ 延迟调拨状态

  • → 待配货 (重新配货-采购完成)
  • → 配货关闭 (关闭需求-无法补货)
  • → 延迟调拨 (转交任务)
  • → 配货退回 (审批回退)

🛒 采购中状态

  • → 配货关闭 (关闭需求-采购不足)
  • → 采购中 (转交任务)
  • → 配货退回 (审批回退)

🔄 转交规则

  • ✅ 待配货 (周边综合支持/管理员)
  • ✅ 延迟调拨 (周边综合支持/管理员)
  • ✅ 采购中 (周边综合支持/管理员)

🚫 关闭规则

  • 📋 待配货 (单品无法完成/不合规)
  • ⏰ 延迟调拨 (确认无法补货)
  • 🛒 采购中 (采购数量不足)
